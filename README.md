## TL;DR

**dojo.tech** is a proactive business intelligence engine built on top of Dojo payments data. It turns transactions into plain-English growth insights for merchants — via a dashboard, a chatbot, and proactive alerts.

**Three hard rules**:
1. Never surface card numbers, CVVs, or customer PII
2. Never write more than two short paragraphs per insight
3. Never call it an AI — it's the Dojo intelligence engine

**Brand**: Active Teal `#3CEAC7` · Serene Green `#003F33` · DOJO Roobert typeface · Bento box graphic system · SME voice register
